# CREOSIGIL

> **의도를 상징으로 완성하다**

CREOSIGIL은 사용자가 자신의 의도와 바람을 상징적 문양(시길, Sigil)으로 만들고, 고화질 PNG로 저장해 배경화면이나 다른 앱에서 활용할 수 있게 하는 모바일 전용 PWA 앱입니다.

---

## ✨ 주요 기능

| 기능 | 내용 |
|---|---|
| **시길 소개** | 앱 첫 화면에서 시길의 의미와 앱 철학 안내 |
| **의도 선택** | 8가지 목적 선택 (보호, 정화, 번영, 집중, 회복, 차단, 인연, 직관) |
| **전통형 시길** | 원형인장 / 마법진 / 봉인문 / 룬결합 / 의식대칭 / 월상결속 |
| **현대형 시길** | 미니멀로고 / 기하학 / 네온심볼 / 엠블럼 / 추상 / 크리스탈 |
| **8가지 분위기** | 흑요석 다크, 월광 실버, 심홍 봉인, 백색 정화, 황동 각인, 네온 아스트랄, 미니멀 럭스, 코스믹 블루 |
| **단계별 조언** | 각 선택 단계마다 짧은 조언 + '더 알아보기' 확장 설명 |
| **필수 선택 검증** | 모든 옵션을 선택해야 다음 단계로 진행 가능 |
| **생성 진행률** | 시길 생성 중 % 진행률 실시간 표시 |
| **고화질 저장** | PNG (1080×1920 / 1440×2560 / 2160×3840) |
| **투명 PNG** | 다른 앱 오버레이 활용용 |
| **SVG 저장** | 벡터 형식 저장 |
| **공유** | Web Share API로 다른 앱에 직접 전달 |
| **보관함** | 최근 20개 저장, 다시 편집 가능 |
| **PWA** | 홈 화면 설치, 오프라인 지원 |

## 📱 화면 흐름

```
인트로 (시길 소개) 
  ↓
홈 (빠른 제작 / 정교한 제작)
  ↓
1단계: 목적 선택 (최대 3개, 1개 권장)
  ↓
2단계: 전통형 / 현대형 선택
  ↓
3단계: 세부 옵션 선택 (필수 전체 선택)
  ↓
4단계: 분위기 선택
  ↓
편집 (크기/회전/광채 조절, 재생성)
  ↓
저장 & 공유 (해상도 선택, PNG/투명PNG/SVG/공유)
  ↓
보관함 저장
```

## 🎨 디자인 특징

- **폰트**: Cinzel Decorative (앱 타이틀), Cinzel (섹션 제목), Noto Serif KR (본문)
- **배색**: 흑요석 배경 (#0A0A0F) + 보라빛 글로우 (#9B8CFF) + 라벤더 라인 (#D9CFFF)
- **무드**: 다크 오컬트 + 미니멀 럭셔리

## 🛠 기술 스택

| 항목 | 내용 |
|---|---|
| 언어 | HTML5, CSS3, JavaScript (ES6+) |
| 그래픽 | Canvas API |
| 저장 | LocalStorage, IndexedDB (예정) |
| PWA | manifest.json, Service Worker |
| 공유 | Web Share API |
| 폰트 | Google Fonts (Cinzel Decorative, Cinzel, Noto Serif KR) |
| 외부 의존성 | 없음 (순수 바닐라 JS) |
