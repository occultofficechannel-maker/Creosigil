# CREOSIGIL

> **의도를 상징으로 완성하다**  
> 나만의 시길(Sigil)을 제작하고 고화질 배경화면으로 저장하는 모바일 전용 PWA

---

## 앱 개요

**CREOSIGIL**은 사용자가 의도를 이해하고 선택하면서 전통형 또는 현대형 시길을 제작하고, 고화질 PNG·투명 PNG·SVG로 저장해 배경화면이나 다른 앱에서 재사용할 수 있는 모바일 전용 PWA입니다.

### 앱 이름 유래
- **Creo** (라틴어 — 만들다, 창조하다)
- **Sigil** (라틴어 sigillum — 인장, 봉인)
- → *나는 시길을 만든다*

---

## 주요 기능

| 기능 | 내용 |
|---|---|
| 시길 소개 | "시길이란 무엇인가?" 설명 + 더보기 |
| 목적 선택 | 8가지 목적 + 동적 조언 (1개/2개/3개 선택별) |
| 전통형 시길 | 원형인장/마법진/봉인문/룬결합/의식대칭/월상결속 |
| 현대형 시길 | 미니멀로고/기하학/네온심볼/엠블럼/추상/크리스탈 |
| 분위기 테마 | 8가지 테마 (흑요석/월광/심홍/백색/황동/네온/미니멀/코스믹) |
| 세부 옵션 검증 | 모든 필수 선택이 완료되어야 다음 단계로 이동 가능 |
| 생성 진행률 | 시길 생성 중 % 표시 및 진행바 |
| 편집 | 크기/광채/선 굵기/색상/배경 실시간 조절 |
| 저장 | 배경화면 PNG(HD/QHD/4K) / 투명 PNG / SVG |
| 공유 | OS 공유 시트 (Web Share API) |
| 보관함 | 최근 20개 저장, 재편집 가능 |
| QR 생성기 | 홈 화면에서 바로 QR 코드 생성 및 저장 |
| 유튜브 링크 | 홈 화면에서 @cursekey 채널로 바로 이동 |
| PWA | 홈 화면 설치, 오프라인 지원 |

## 화면 구조

```
홈 화면
  ├── QR 생성기 버튼
  ├── YouTube (@cursekey) 버튼
  ├── 시길 소개 카드
  ├── 앱 철학 카드
  ├── 최근 보관함 그리드
  └── 시길 만들기 시작 버튼

의도 목적 선택 (STEP 01)
  └── 8가지 목적 선택 (복수 선택 가능, 1개 권장)

시길 유형 선택 (STEP 02)
  └── 전통형 / 현대형

세부 옵션 (STEP 03)
  ├── 전통형: 봉인구조/중심상징/밀도/선스타일/대칭/외곽장식
  └── 현대형: 표현스타일/선스타일/복잡도/대칭

분위기 선택 (STEP 04)
  └── 8가지 분위기 테마

생성 & 편집 (STEP 05)
  └── 실시간 편집 (크기/광채/색상/배경)

저장 & 공유
  ├── 배경화면 PNG (HD/QHD/4K)
  ├── 투명 PNG
  ├── SVG
  ├── 공유하기
  └── 보관함으로 가기
```

## 기술 스택

- **HTML/CSS/JavaScript** (Vanilla, 프레임워크 없음)
- **Canvas API** — 시길 생성 및 렌더링
- **Web Share API** — 이미지 공유
- **localStorage** — 보관함 저장
- **QRCode.js** (CDN) — QR 코드 생성
- **Google Fonts** — Cinzel Decorative, Cinzel, Noto Serif KR
- **PWA** — manifest.json + Service Worker

## 출력 옵션

| 포맷 | 용도 |
|---|---|
| PNG (배경 포함) | 배경화면으로 직접 사용 |
| 투명 PNG | 다른 앱에 레이어로 사용 |
| SVG | 크기 제한 없는 벡터 파일 |

### 해상도 프리셋
- HD: 1080 × 1920
- QHD: 1440 × 2560 (기본)
- 4K: 2160 × 3840
